# [Advent of Code](https://www.adventofcode.com) 2025 Solutions in rust

These are my personal solutions for AOC 2025 written in rust. I am using this years AOC as a way to get more practive with coding problems in rust. I can't guarantee the solutions are correct, well written, optimal, or idomatic rust.

## Running Each Day
- Add the input.txt to the respective day folder in the input folder
- Run `cargo run --release 'day(s)'` where 'day(s)' is the day(s) you want to run
- e.g., `cargo run --release 1 2 4`
